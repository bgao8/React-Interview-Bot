## Hone

Hone is an AI-powered interview practice platform built with a React frontend and a Python backend.
Its goal is to make interview preparation more realistic, structured, and feedback-driven.

The project was motivated by my own experience preparing for interviews, where practicing answers alone or reading prompKey Featurests felt disconnected from the pressure and pacing of real conversations.
I wanted a tool that could simulate an actual interview environment, listen to spoken responses, and provide intelligent follow-up and evaluation in real time.
Hone was built to bridge the gap between passive preparation and active interview performance.

This project represents my first end-to-end React application, paired with a Python backend for AI and audio processing.
I built Hone to learn modern frontend development patterns while designing a backend pipeline responsible for transcription, prompt generation, and response evaluation.

The focus was on building a production-style, full-stack system that tightly integrates user interaction with AI-driven logic.

## Key Features
**Custom interview configuration**:
- Role, level, company, and interview type selection
**Live interview simulation**
- AI-generated interview questions
- Spoken responses via microphone input
**Real-time audio capture and transcription**
**Python-based AI pipeline**, including
- Speech-to-text processing
- Context-aware follow-up questions
- Answer evaluation and feedback
